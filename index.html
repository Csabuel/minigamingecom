Product Description

Power up your productivity, gaming, and content creation with the Intel Core i9-13900KF 3 GHz 24-Core LGA 1700 Processor. This 13th generation desktop processor builds upon the Hybrid Core architecture and Intel 7 process by fitting 24 cores— eight more than the previous generation— onto a single processor while still fitting the LGA 1700 socket.
